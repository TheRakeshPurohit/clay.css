clay.css

Micro CSS util class for applying inflated fluffy 3D claymorphic styles to elements. Fully customizable and extensible with CSS variables and SASS mixin for even more styling options.

Read the docs Support project

1. Get it

NPM

npm i claymorphism-css
yarn add claymorphism-css

CDN

<link
  rel="stylesheet"
  href="https://unpkg.com/claymorphism-css/dist/clay.css"
/>

2. Apply it

Class clay provides bare minimum styles to apply the claymorphic effect to any HTML element. Additional styles (layout, position, colors, borders, etc.) should be added by using an additional CSS class.

HTML

                
                  <div class="clay">

                      Fluffy ipsum dolor sit amet consectetur...

                  </div>
